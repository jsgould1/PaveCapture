Keyboard Mode Setup:
1. On the DISTO X3: Settings → Bluetooth → set to Keyboard mode
2. Pair the DISTO with Windows via Settings → Bluetooth
3. Click the input field below
4. Shoot with the DISTO — the value types itself in
5. The DISTO sends Enter automatically — measurement is logged

Tap A Next before shooting to auto-tag as pavement.
Tap B Next before shooting to auto-tag as shoulder.
The drop-off calculation appears automatically when both are set.
